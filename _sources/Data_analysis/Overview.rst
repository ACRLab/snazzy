
Overview
========

The results from processing the raw data with the ``snazzy_processing`` package are analyzed with ``snazzy_analysis``.
The analysis can be performed using the provided graphical user interface.

The main functionalities provided are:

* Calculate ΔF/F traces
* Quantify peaks and metrics from ΔF/F traces
* Data visualization, comparison, and curation
